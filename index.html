🎵 Spotify Analytics Dashboard

An interactive Spotify Analytics Dashboard built using Power BI to analyze songs, artists, albums and popularity through dynamic visualizations.

Download PBIX Dataset GitHub

📌 Project Overview

This project recreates my earlier Spotify Analytics Dashboard built in Excel, this time using Power BI with interactive navigation, Power Query transformations, DAX measures and dynamic visuals.

✨ Features

  • Interactive multi-page navigation
  • Dynamic artist selection
  • KPI Cards
  • Interactive charts & slicers
  • Power Query transformations
  • DAX Measures

🛠 Tech Stack

Power BI • Power Query • DAX • Data Modeling • Data Visualization
